# OTT Movie Trailer Website

A responsive movie trailer website developed as an academic project done 7 years before.

## Features
- Movie listings
- Upcoming movies
- Trailer pages
- User registration and login UI
- Movie booking page
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
